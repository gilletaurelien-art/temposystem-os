# Journal de Bord

Ce dossier concentre la mémoire longue de TEMPOSYSTEM OS. Il documente la vision, les principes, l'architecture, les décisions et les apprentissages du projet.

## Fichiers

- `00_VISION.md` : intention générale et horizon produit.
- `01_PRINCIPES.md` : règles de conception et de gouvernance.
- `02_ARCHITECTURE.md` : structure technique et flux prévus.
- `03_DOMAINES.md` : périmètres fonctionnels.
- `04_ROADMAP.md` : jalons à court et moyen terme.
- `05_DECISIONS.md` : décisions structurantes.
- `06_JOURNAL.md` : notes chronologiques.
- `07_LESSONS_LEARNED.md` : apprentissages réutilisables.
- `08_AI_CONTEXT.md` : contexte destiné aux rôles et à leurs implémentations IA.
- `09_BACKLOG.md` : idées, tâches et dettes.
- `10_GLOSSAIRE.md` : vocabulaire partagé.
