# Principes

- Le Capitaine décide.
- Les membres du Conseil éclairent, ils ne tranchent pas seuls.
- Les rôles sont permanents, les implémentations sont interchangeables.
- Chaque décision importante doit laisser une trace.
- La mémoire doit rester lisible par un humain.
- Le système avance par petits modules utiles.
- Aucun connecteur externe n'est ajouté avant d'avoir un usage clair.
